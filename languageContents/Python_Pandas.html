세부 내용

  1. 데이터 구조
    1. 데이터프레임 (DataFrame)
      • 정의: 2차원 레이블링된 데이터 구조
      • 구성 요소:
        • 인덱스(Index): 행 식별자
        • 컬럼(Columns): 열 식별자
        • 데이터(Data): 실제 저장된 값
      • 생성 방법:
        • 딕셔너리로 생성: pd.DataFrame(dict)
        • 리스트로 생성: pd.DataFrame(list)
        • 파일에서 생성: pd.read_csv(), pd.read_excel()
    2. 시리즈 (Series)
      • 정의: 1차원 레이블링된 배열
      • 구성 요소:
        • 인덱스(Index): 데이터 식별자
        • 데이터(Data): 실제 저장된 값
  2. 데이터 조작
    1. 데이터 탐색
      • 기본 정보 확인
        • head(): 상위 n개 행 확인
        • tail(): 하위 n개 행 확인
        • info(): 데이터프레임 정보 확인
        • describe(): 기초 통계량 확인
    2. 데이터 전처리
      • 결측치 처리
        • isna(): 결측치 확인
        • fillna(): 결측치 채우기
        • dropna(): 결측치 제거
      • 중복 처리
        • duplicated(): 중복 확인
        • drop_duplicates(): 중복 제거

  1. 직관적인 데이터 구조와 조작 방법 제공

## 청크 단위 처리
def process_large_file(filename):
    chunk_size = 1000
    chunks = []
    for chunk in pd.read_csv(filename, chunksize=chunk_size):
        # 데이터 처리 예시
        processed = chunk.dropna()  # 결측치 제거
        chunks.append(processed)
    return pd.concat(chunks)

## 메모리 최적화
def optimize_dataframe(df):
    # 데이터 타입 최적화
    for col in df.columns:
        if df[col].dtype == 'object':
            df[col] = df[col].astype('category')
        elif df[col].dtype == 'float64':
            df[col] = pd.to_numeric(df[col], downcast='float')
        elif df[col].dtype == 'int64':
            df[col] = pd.to_numeric(df[col], downcast='integer')
    return df
