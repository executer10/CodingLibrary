세부 내용 정의

  1. print()
    1. 개념 설명
      • print() 함수는 출력합니다.
      • 예시: print('Hello') -> Hello 출력
  2. input()
    1. 개념 설명
      • input() 함수는 사용자 입력을 받습니다.
      • 예시: input('이름을 입력하세요: ') -> 사용자 입력 값
  3. len()
    1. 개념 설명
      • len() 함수는 객체의 길이를 반환합니다.
      • 예시: len([1, 2, 3]) -> 3
  4. type()
    1. 개념 설명
      • type() 함수는 객체의 타입을 반환합니다.
      • 예시: type(5) -> <class 'int'>
  5. str()
    1. 개념 설명
      • str() 함수는 객체를 문자열로 변환합니다.
      • 예시: str(10) -> '10'
  6. int()
    1. 개념 설명
      • int() 함수는 문자열이나 숫자를 정수로 변환합니다.
      • 예시: int('10') -> 10
  7. float()
    1. 개념 설명
      • float() 함수는 숫자나 문자열을 부동소수점 숫자로 변환합니다.
      • 예시: float('3.14') -> 3.14
  8. bool()
    1. 개념 설명
      • bool() 함수는 객체를 불리언 값으로 변환합니다.
      • 예시: bool(1) -> True
  9. list()
    1. 개념 설명
      • list() 함수는 리스트 객체를 생성합니다.
      • 예시: list((1, 2, 3)) -> [1, 2, 3]
  10. tuple()
    1. 개념 설명
      • tuple() 함수는 튜플 객체를 생성합니다.
      • 예시: tuple([1, 2, 3]) -> (1, 2, 3)
  11. set()
    1. 개념 설명
      • set() 함수는 집합 객체를 생성합니다.
      • 예시: set([1, 2, 3]) -> {1, 2, 3}
  12. dict()
    1. 개념 설명
      • dict() 함수는 새로운 사전 객체를 생성합니다.
      • 예시: dict(a=1, b=2) -> {'a': 1, 'b': 2}
  13. abs()
    1. 개념 설명
      • abs() 함수는 숫자의 절대값을 반환합니다.
      • 예시: abs(-5) -> 5
  14. min()
    1. 개념 설명
      • min() 함수는 인수 중 가장 작은 값을 반환합니다.
      • 예시: min(1, 2, 3) -> 1
  15. max()
    1. 개념 설명
      • max() 함수는 인수 중 가장 큰 값을 반환합니다.
      • 예시: max(1, 2, 3) -> 3
  16. sum()
    1. 개념 설명
      • sum() 함수는 숫자들의 합을 반환합니다.
      • 예시: sum([1, 2, 3]) -> 6
  17. sorted()
    1. 개념 설명
      • sorted() 함수는 정렬된 리스트를 반환합니다.
      • 예시: sorted([3, 1, 2]) -> [1, 2, 3]
  18. range()
    1. 개념 설명
      • range() 함수는 숫자 시퀀스를 생성합니다.
      • 예시: range(5) -> range(0, 5)
  19. enumerate()
    1. 개념 설명
      • enumerate() 함수는 인덱스와 요소를 포함하는 이터레이터를 반환합니다.
      • 예시: enumerate(['a', 'b']) -> (0, 'a'), (1, 'b')
  20. zip()
    1. 개념 설명
      • zip() 함수는 여러 이터러블을 묶어줍니다.
      • 예시: zip([1, 2], ['a', 'b']) -> [(1, 'a'), (2, 'b')]
  21. map()
    1. 개념 설명
      • map() 함수는 함수와 이터러블을 인수로 받아, 함수가 적용된 결과를 반환합니다.
      • 예시: map(str, [1, 2, 3]) -> ['1', '2', '3']
  22. filter()
    1. 개념 설명
      • filter() 함수는 조건에 맞는 요소를 걸러냅니다.
      • 예시: filter(lambda x: x > 0, [-1, 0, 1]) -> [1]
  23. all()
    1. 개념 설명
      • all() 함수는 모든 요소가 참이면 True를 반환합니다.
      • 예시: all([True, True, False]) -> False
  24. any()
    1. 개념 설명
      • any() 함수는 하나 이상의 요소가 참이면 True를 반환합니다.
      • 예시: any([False, True, False]) -> True
  25. round()
    1. 개념 설명
      • round() 함수는 숫자를 반올림합니다.
      • 예시: round(3.14159, 2) -> 3.14
  26. pow()
    1. 개념 설명
      • pow() 함수는 거듭제곱을 계산합니다.
      • 예시: pow(2, 3) -> 8
  27. divmod()
    1. 개념 설명
      • divmod() 함수는 몫과 나머지를 튜플로 반환합니다.
      • 예시: divmod(7, 3) -> (2, 1)
  28. hex()
    1. 개념 설명
      • hex() 함수는 정수를 16진수 문자열로 변환합니다.
      • 예시: hex(255) -> '0xff'
  29. oct()
    1. 개념 설명
      • oct() 함수는 정수를 8진수 문자열로 변환합니다.
      • 예시: oct(8) -> '0o10'
  30. bin()
    1. 개념 설명
      • bin() 함수는 정수를 2진수 문자열로 변환합니다.
      • 예시: bin(2) -> '0b10'
  31. chr()
    1. 개념 설명
      • chr() 함수는 유니코드 값을 문자로 변환합니다.
      • 예시: chr(97) -> 'a'
  32. ord()
    1. 개념 설명
      • ord() 함수는 문자의 유니코드 값을 반환합니다.
      • 예시: ord('a') -> 97
  33. id()
    1. 개념 설명
      • id() 함수는 객체의 고유 식별자를 반환합니다.
      • 예시: id(obj) -> 객체의 메모리 주소
  34. dir()
    1. 개념 설명
      • dir() 함수는 객체의 속성과 메서드 목록을 반환합니다.
      • 예시: dir([]) -> 리스트 객체의 속성과 메서드 목록
  35. help()
    1. 개념 설명
      • help() 함수는 객체에 대한 도움말을 제공합니다.
      • 예시: help(print) -> print 함수에 대한 설명 출력
  36. eval()
    1. 개념 설명
      • eval() 함수는 문자열로 표현된 코드를 실행합니다.
      • 예시: eval('1 + 2') -> 3
  37. exec()
    1. 개념 설명
      • exec() 함수는 동적으로 파이썬 코드를 실행합니다.
      • 예시: exec('print("Hello")') -> Hello 출력
  38. open()
    1. 개념 설명
      • open() 함수는 파일을 엽니다.
      • 예시: open('file.txt', 'r') -> 파일 객체
  39. callable()
    1. 개념 설명
      • callable() 함수는 객체가 호출 가능한지 확인합니다.
      • 예시: callable(print) -> True
  40. isinstance()
    1. 개념 설명
      • isinstance() 함수는 객체가 특정 클래스의 인스턴스인지 확인합니다.
      • 예시: isinstance(5, int) -> True
  41. issubclass()
    1. 개념 설명
      • issubclass() 함수는 클래스가 특정 클래스의 서브클래스인지 확인합니다.
      • 예시: issubclass(bool, int) -> True
  42. getattr()
    1. 개념 설명
      • getattr() 함수는 객체의 속성 값을 반환합니다.
      • 예시: getattr(obj, 'attr') -> obj.attr
  43. setattr()
    1. 개념 설명
      • setattr() 함수는 객체의 속성 값을 설정합니다.
      • 예시: setattr(obj, 'attr', 10) -> obj.attr = 10
  44. hasattr()
    1. 개념 설명
      • hasattr() 함수는 객체가 특정 속성을 가지고 있는지 확인합니다.
      • 예시: hasattr(obj, 'attr') -> True 또는 False
  45. delattr()
    1. 개념 설명
      • delattr() 함수는 객체의 속성을 삭제합니다.
      • 예시: delattr(obj, 'attr') -> obj.attr 삭제
  46. property()
    1. 개념 설명
      • property() 함수는 속성을 정의합니다.
      • 예시: property(fget=None, fset=None, fdel=None, doc=None) -> 속성 객체
  47. staticmethod()
    1. 개념 설명
      • staticmethod() 함수는 정적 메서드를 정의합니다.
      • 예시: staticmethod(func) -> 정적 메서드
  48. classmethod()
    1. 개념 설명
      • classmethod() 함수는 클래스 메서드를 정의합니다.
      • 예시: classmethod(func) -> 클래스 메서드
  49. super()
    1. 개념 설명
      • super() 함수는 부모 클래스의 메서드를 호출합니다.
      • 예시: super().method() -> 부모 클래스의 method 호출
  50. object()
    1. 개념 설명
      • object() 함수는 모든 클래스의 기본 클래스입니다.
      • 예시: obj = object() -> 새로운 객체 생성
  51. bytes()
    1. 개념 설명
      • bytes() 함수는 바이트 객체를 생성합니다.
      • 예시: bytes(5) -> 바이트 객체
  52. bytearray()
    1. 개념 설명
      • bytearray() 함수는 바이트 배열 객체를 생성합니다.
      • 예시: bytearray(5) -> 바이트 배열 객체
  53. memoryview()
    1. 개념 설명
      • memoryview() 함수는 바이트 데이터를 메모리 뷰 객체로 변환합니다.
      • 예시: memoryview(b'abc') -> 메모리 뷰 객체
  54. frozenset()
    1. 개념 설명
      • frozenset() 함수는 변경 불가능한 집합을 생성합니다.
      • 예시: frozenset([1, 2, 3]) -> frozenset({1, 2, 3})
  55. complex()
    1. 개념 설명
      • complex() 함수는 복소수를 생성합니다.
      • 예시: complex(1, 2) -> (1+2j)
  56. ascii()
    1. 개념 설명
      • ascii() 함수는 객체의 ASCII 표현을 반환합니다.
      • 예시: ascii('파이썬') -> "'\\ud30c\\uc774\\uc36c'"
  57. repr()
    1. 개념 설명
      • repr() 함수는 객체의 문자열 표현을 반환합니다.
      • 예시: repr(10) -> '10'
  58. globals()
    1. 개념 설명
      • globals() 함수는 전역 심볼 테이블의 사전을 반환합니다.
      • 예시: globals() -> 전역 변수 사전
  59. locals()
    1. 개념 설명
      • locals() 함수는 현재 로컬 심볼 테이블의 사전을 반환합니다.
      • 예시: locals() -> 로컬 변수 사전
  60. vars()
    1. 개념 설명
      • vars() 함수는 객체의 __dict__ 속성을 반환합니다.
      • 예시: vars() -> 현재 로컬 심볼 테이블의 사전
  61. compile()
    1. 개념 설명
      • compile() 함수는 소스 코드를 코드 객체로 컴파일합니다.
      • 예시: compile('print(1)', '', 'exec') -> 코드 객체
  62. __import__()
    1. 개념 설명
      • __import__() 함수는 모듈을 동적으로 가져옵니다.
      • 예시: __import__('os') -> os 모듈
  63. next()
    1. 개념 설명
      • next() 함수는 이터레이터의 다음 항목을 반환합니다.
      • 예시: next(iter([1, 2, 3])) -> 1
  64. iter()
    1. 개념 설명
      • iter() 함수는 이터레이터 객체를 반환합니다.
      • 예시: iter([1, 2, 3]) -> 이터레이터 객체
  65. reversed()
    1. 개념 설명
      • reversed() 함수는 시퀀스를 역순으로 반환합니다.
      • 예시: reversed([1, 2, 3]) -> [3, 2, 1]
  66. slice()
    1. 개념 설명
      • slice() 함수는 슬라이스 객체를 생성합니다.
      • 예시: slice(1, 5, 2) -> 슬라이스 객체
  67. format()
    1. 개념 설명
      • format() 함수는 문자열을 포맷합니다.
      • 예시: format(3.14159, '.2f') -> '3.14'
  68. hash()
    1. 개념 설명
      • hash() 함수는 객체의 해시 값을 반환합니다.
      • 예시: hash('abc') -> 해시 값

특징

  1. 파이썬 내장 함수는 다양한 데이터 타입을 지원합니다.
  2. 함수 호출이 간단하고 직관적입니다.

사용 이유

  1. 코드의 가독성을 높이고 유지보수를 용이하게 합니다.
  2. 복잡한 연산을 간단하게 처리할 수 있습니다.

# isinstance() 예제
print(isinstance(5, int))  # True

# issubclass() 예제
print(issubclass(bool, int))  # True

# getattr() 예제
class Example:
    attr = 10
print(getattr(Example, 'attr'))  # 10

# setattr() 예제
class Example:
    pass
obj = Example()
setattr(obj, 'attr', 10)
print(obj.attr)  # 10

# hasattr() 예제
class Example:
    attr = 10
print(hasattr(Example, 'attr'))  # True

# delattr() 예제
class Example:
    attr = 10
delattr(Example, 'attr')
print(hasattr(Example, 'attr'))  # False

# property() 예제
class C:
    def __init__(self):
        self._x = None
    def getx(self):
        return self._x
    def setx(self, value):
        self._x = value
    def delx(self):
        del self._x
    x = property(getx, setx, delx, "I'm the 'x' property.")
c = C()
c.x = 5
print(c.x)  # 5
