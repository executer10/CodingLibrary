구분 저장되는 값에 따른 분류 타입의 종류
기본타입 정수 타입 byte, char, short, int, long
실수 타입 float, double
논리 타입 boolean
정수타입
타입 메모리 사용 크기 저장되는 값의 허용 범위
byte 1byte 8bit -2⁷ ~ (2⁷-1) -128 ~ 127
short 2byte 16bit -2¹⁵~ (2¹⁵-1) -32768 ~ 32767
char 2byte 16bit 0 ~ (2¹⁶-1) 0~65535(유니코드)
int 4byte 32bit -2³¹ ~ (2³¹-1) -2,147,484,648 ~ 2,147,484,647
long 8byte 64bit -2⁶³ ~ (2⁶³-1) -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807
String 타입
타입 메모리 사용 크기 저장되는 값의 허용 범위
String 문자열 길이에 따라 가변적 문자열(문자의 나열)
실수 타입
타입 메모리 사용 크기 저장되는 값의 허용 범위
float 4byte(32bit) ±(1.40129846432481707e-45 ~ 3.40282346638528860e+38)
double 8byte(64bit) ±(4.94065645841246544e-324d ~ 1.79769313486231570e+308d)
논리 타입
타입 메모리 사용 크기 저장되는 값의 허용 범위
boolean 1byte(8bit) true, false
